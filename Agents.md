# Se++ Repository Context

Purpose: Root workspace for Se++ (Session++), an AI-powered exam and study trainer.

Rules:
- Keep frontend code in `apps/web`.
- Keep Cloudflare Worker API code in `workers/api`.
- Keep portable contracts and shared logic in `shared`.
- Add or update a local `Agents.md` whenever a major module is created or its public responsibilities change.
- UI copy must be English.
